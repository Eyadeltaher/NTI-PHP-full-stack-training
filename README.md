**PHP Full Stack Training (Backend-Focused)**

This repository documents my journey through a PHP Full Stack training program with a strong emphasis on backend development.

The training covers the fundamentals of web development, starting with basic frontend technologies (HTML & CSS) to build simple user interfaces, followed by a deeper focus on backend engineering using PHP.

Key areas include:

* Core PHP programming and OOP concepts
* Building dynamic web applications
* Handling HTTP requests and server-side logic
* Database integration (MySQL)
* Authentication and session management
* Writing secure and maintainable backend code

The goal of this training is to gain a solid understanding of how both frontend and backend systems work together, with a deeper focus on backend behavior. This knowledge supports my work in security engineering by helping me better understand application logic, identify vulnerabilities, and analyze the context in which they occur.
